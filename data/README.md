# Data

This folder contains the raw and processed datasets used in the project.

---

## Raw Datasets

Place the following two files in `data/raw/` before running any scripts.


### 1. News Headlines Dataset for Sarcasm Detection

- **Source:** [Kaggle](https://www.kaggle.com/datasets/rmisra/news-headlines-dataset-for-sarcasm-detection)
- **File name:** `Sarcasm_Headlines_Dataset.json`
- **Domain:** News headlines (The Onion — sarcastic, HuffPost — normal)
- **Size:** 26,709 rows
- **Format:** Newline-delimited JSON with three fields: `headline`, `is_sarcastic`, `article_link`

### 2. Forum Comments Dataset (GEN-sarc-notsarc)

- **Source:** [Kaggle](https://www.kaggle.com/datasets/coldn00ldes/sarcasm-corpus-v2oraby-et-al) — Sarcasm Corpus V2 (Oraby et al., 2016)
- **File name:** `GEN-sarc-notsarc.csv`
- **Domain:** Usenet forum comments
- **Size:** 6,520 rows (50% sarcastic, 50% normal)
- **Format:** CSV with three fields: `class`, `id`, `text`

---

## Processed Datasets

These files are generated automatically by the scripts. Do not edit manually.

| File | Generated by | Description |
|---|---|---|
| `processed/cleaned_sarcasm_data.csv` | `02_preprocess_data.py` | Cleaned news headlines with 6 engineered features |
| `processed/cleaned_gen_data.csv` | `05_preprocess_gen.py` | Cleaned forum comments with 6 engineered features |
| `processed/combined_data.csv` | `05b_combine_datasets.py` | News + forum merged, shuffled, with `source` column |
